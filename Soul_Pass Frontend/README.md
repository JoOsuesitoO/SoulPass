# Frontend de SoulPass

Toda esta carpeta es el frontend de nuestro proyecto SoulPass. Este front está realizado con react y typescript, para una facilidad de uso e intuicion. Se realizo en base a un mock up desarrollado en figma.

Para iniciar, primero tenemos que instalar las dependencias en la consola o terminal con:
npm install

Una vez instaladas las dependencias, podemos inicar el programa con:
npm run dev
